# Hello-World-

Hello World – My First Repository

PROJECT TITLE

Hello World – Python Sample Project

DESCRIPTION

This repository was created to practice using GitHub and building a properly formatted README.md file.
In this project, I created a simple Python program that prints "Hello, World!" to the console. The main goal was to:
Learn how to create a repository,
Add and edit a README file,
and use multiple Markdown formatting options,
Practice organizing project documentation.
This project demonstrates my understanding of basic Python structure and GitHub workflow.

TOOLS USED:

Python

GitHub

Markdown

Visual Studio Code (or any code editor)

FILES USED:

Hello_World/

│── README.md

│── hello_world.py

File Descriptions

README.md – Project documentation

hello_world.py – Python script that prints a message

PYTHON PROGRAM:

print("Hello, World!")

HOW TO RUN THE PROGRAM:

Make sure Python is installed on your computer.
Download or clone this repository.
Open a terminal in the project folder.
Run the following command:
python hello_world.py
You should see:
Hello, World!
